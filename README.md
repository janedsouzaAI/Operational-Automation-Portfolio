## Overview
This portfolio showcases automation and data management projects designed to streamline fund onboarding, improve operational efficiency, and reduce errors across trading and compliance teams.

---

## Projects

### 1. Broker-Fund Mapping Tool
- **Problem:** Manual tracking of broker-fund relationships caused errors and delays.  
- **Solution:** Automated Python-based report to track mappings accurately.  
- **Impact:** Reduced booking errors and saved time for trading teams.

### 2. Onboarding Lifecycle Tracker
- **Problem:** Excel-based tracking was inefficient across multiple teams.  
- **Solution:** Appsmith + MongoDB tool managing onboarding lifecycle for liquid and illiquid funds.  
- **Impact:** Real-time KPI visibility, improved coordination, faster onboarding.

### 3. Power BI KPI Dashboards
- **Purpose:** Visualize onboarding progress and operational performance.  
- **Impact:** Supported management decisions with clear insights.
